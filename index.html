蚂蚁集团-国际事业群-全球技术【春季2023届校园招聘内推投递指南】

发表于 2022-03-01

亲爱的同学，欢迎参加蚂蚁集团-国际事业群-全球技术春季 2023 届校园招聘的内推活动！
以下是投递流程的操作说明，方便你了解~

【我们是谁？】

蚂蚁集团服务商家和消费者实现“全球收”、“全球付”、“全球汇”。通过创新的全球跨境支付解决方案，用前沿的移动支付技术构建跨境支付能力，助力全球商家一次性接入多种数字支付方式，和全球的消费者建立连接；同时也与全球科技公司、金融机构和商户广泛合作，在本地合规展业，在数字支付、数字营销、贸易金融、数字银行等领域分享技术和行业专长，助力当地商业数字化转型升级，给世界带来微小而美好的改变。

【我们招什么样的你？】

【面向人群】2023 届毕业生（毕业时间：2022 年 11 月-2023 年 10 月）
投递批次：阿里巴巴 2023 届实习生招聘
【我们在哪里工作？】上海，杭州，成都，深圳、新加坡，地点选择多多，任你选择哦（PS：新加坡只面向当地高校招聘）
【我们招哪些岗位？】如下岗位，同样任你选择哦

  1. 研发工程师 JAVA：(上海、杭州、成都、深圳、新加坡）

我们希望你：来自计算机专业、软件专业或其他相关专业，酷爱计算机以及互联网技术，热衷于解决挑战性的问题，追求极致的用户体验；熟悉数据结构和算法，精通 Java 语言，熟悉 C++，PHP，.NET 等编程语言；如果你在学校作为骨干参与学生网站建设和开发，大学生数学建模竞赛，热衷于 ACM，“挑战杯”，机器人足球比赛等等都加分；

  1. 数据研发工程师：(上海、杭州）

我们希望你：具备比较强的动手能力和学习能力，熟悉至少一门数据处理语言，如 SQL、JAVA、Python、Perl 等，具备开发自定义 UDF 的能力,熟悉 unix 或者 linux 操作；有参与过数据处理、分析、挖掘等相关项目可加分；对 Hadoop、Hive、ODPS、Hbase、spark、flink 等分布式平台有一定的理解也加分；

  1. 算法工程师-机器学习：（上海、杭州、深圳）

我们希望你：理解机器学习、深度学习、强化学习算法原理与应用；在搜索/推荐/时序预测/量化交易/用户画像等至少一个领域有深入研究工作，具备扎实的编程基础和大数据处理经验；

  1. 前端开发工程师:（上海、杭州、成都、深圳）

我们希望你：热爱前端，具备扎实的 HTML、CSS、JS 基础，熟悉常见前端研发框架；理解 Web 标准，熟悉 HTTP 网络协议；熟悉算法、数据结构等计算机科学基础知识；关注技术前沿，在 GitHub 上有细心维护的项目，善于利用各种工具提升自己的开发效率，能够适应前端技术日新月异的环境；加分项：有坚持写技术博客的习惯，有小而美的 PC 或移动端作品，对自己的程序代码有追求

  1. 客户端开发工程师： iOS/Android（上海）

我们希望你：熟悉 iOS/Android 系统机制和各种高级功能有着深刻了解；具备客户端性能优化的经验，有一定软件架构设计能力，熟悉常见的异步、同步、多线程、跨进程、组件、容器的设计方法；重视用户体验积极尝试各种新技术选择最佳实现与我们的产品有效结合，具有优秀的分析和解决实际问题的能力和态度，有创业的激情 ；

  1. 测试开发工程师：（上海、杭州、成都、深圳）

我们希望你：熟悉 C、C++、Java、Python、Perl 等至少一种编程语言，熟悉基础的计算机系统结构、操作系统、网络、分布式系统等知识；掌握软件测试理论和方法，有设计和开发测试工具和自动化测试框架能力更佳；熟悉机器学习算法、自然语言处理、图像算法等优先；有广泛的技术视野，强学习能力和解决问题的能力；

  1. 技术支持工程师：（上海、杭州）

我们希望你：熟悉 Linux 和 Windows 系统及常见服务的配置及使用管理（如 Apache、IIS、NGINX 等），了解计算机网络及 TCP/IP 协议，具有一定的系统故障排查能力优先，具备良好的沟通能力和团队协同能力，客户服务意识，良好的责任心，跟进解决客户的问题，优化系统、改善流程、增强客户和 CP 体验。

【如何加入我们，获得师兄的精准内推？】

你会有 2 种方式参与内推：
方式一-直接内推：将你的简历（如果没有最新简历，提供手机、姓名、邮箱即可）、意向岗位提供给阿里同学，请 TA 直接在阿里内推系统上，将你内推至对应的岗位，内部跟进你的面试流程。
——优先使用此方式，推进速度会更快！
​

方式二-扫码内推：你收到了阿里同学发送给你的专属内推二维码，收到后，请手机扫描二维码（微信/支付宝/钉钉均可），进入校招申请界面，填写姓名、手机、邮箱并提交。

如果内推人仅指定了部门，未指定岗位，则扫码后会出现该部门下的所有岗位，选择你有意愿的岗位即可。

无论方式一还是方式二，只要申请/内推成功，你都会收到一个短信提示，同时你的邮箱都将会收到一封邮件“诚挚邀请您参加阿里巴巴集团校园招聘内部推荐”

点击邮件中链接，确认意向岗位，请认准“蚂蚁集团-国际事业群-全球技术”为第一意向！
如果推荐岗位非意向岗位，也可以联系内推人重新推荐。

提交简历后，需要做笔试和测评（除前端外的所有技术岗位，均需笔试）。笔试公告
笔试需要在终面前完成，测评在 30 天内均可作答。为了加速你的面试流程，请尽快完成笔试和测评哦。
悄悄告诉你，先完成笔试和测评的同学会被优先安排面试哦！

【如何查看流程进度？】

投递成功后，可以在招聘官网-个人中心-我的应聘，查看自己的流程进度。

如有流程进度上的疑问，建议优先联系内推人，TA 也许能帮助你更快查询到信息。

_# 转换证书格式
_```bash #查看证书
openssl x509 -in cert.crt -text -noout

#查看 DER encoded 证书
openssl x509 -in uhttpd.crt -inform der -text -noout

#PEM to DER
openssl x509 -in cert.crt -outform der -out uhttpd.crt
openssl rsa -in cert.key -outform der -out uhttpd.key

#DER to PEM
openssl x509 -in uhttpd.crt -inform der -outform pem -out cert.crt
openssl rsa -in uhttpd.key -inform der -outform pem -out cert.key

薅美帝企业羊毛-白嫖Oracle cloud 永久免费VPS

发表于 2021-05-30 分类于 折腾

最近在科学工具不太稳定，急需一台 backup 来自动切换以备不时之需。之前听说过 Oracle Cloud VPS 可以永久免费使用，这次尝试白嫖下，没想到一下就成功了。故作次记录

前提条件：

你必须要要有张信用卡，可以是 VSIA，MasterCard 都行，会进行扣款认证。
你可能需要一把梯子，我的移动的网有时候会很访问很慢

注册 & 绑卡

https://signup.cloud.oracle.com/?sourceType=_ref_coc-asset-opcSignIn&language=zh_CN
按正常信息即可，注册，绑卡时我都没得问题（平安车主卡）

创建 vm

进入主页：https://cloud.oracle.com/?region=ap-seoul-1

选择镜像

选择你要有的镜像，Ubuntn Centos 这些都可以

在配置 vm 的时候注意要有这个条件的 _符合始终免费条件 __ ，_1 个核心 OCPU，1 GB 内存，0.48 Gbps 网络带宽(通常只有 50Mbps，每个 VM 是 1/8 的配置，CPU 和带宽都是 1/8)

网络配置

默认即可，如果 IP 时被 Q 的，国内无法访问。

可以通过这个网站检测 https://www.vps234.com/ipchecker 国内 tcp 可以访问即可

如果被 Q 可以重新分配 IP： 在 附加的 VNIC， 查看详细信息里面可修改

登陆 vm

在创建 vm 的时候需要保持 ssh 密钥或者自己上传已有的密钥，否则你无法远程登陆

oracle cloud vm 首次无法远程 root 账户登陆
如果 vm 是 Linux, CentOS, or Windows 你的用户名为 opc
如果 vm 是 Ubuntu 你的用户名是 ubuntu
使用密钥登陆：ubuntu@ip or opc@ip

参考文档：https://docs.oracle.com/en-us/iaas/Content/Compute/Tasks/accessinginstance.htm#Prerequi

DQ77KB-AMT-reset-password

发表于 2020-05-10 更新于 2021-05-30

之前有一块板子做软路由，闲置很久了，导致 intl AMT 密码遗忘，找了很久的资料终于重置成功了

  1. 断掉电源
  2. 找到主板跳线
  3. 移动主板跳线
  4. 通电开机进入设置
  5. Reset Intel AMT
  6. 关机，断电
  7. 还原跳线，通电开机，Ctrl + P 即可进入 AMT 输入默认密码即可
  8. 默认密码有 admin
