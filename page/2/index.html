WSL Ubuntu 编译K3 syncdial 单线多拨

发表于 2018-01-20 更新于 2021-05-30 分类于 折腾

K3刷了LEDE想单线多拨，发现软件源内没有syncdial。手动设置感觉麻烦(没设置过啊，一键多好)

编译需要在Linux环境下，虚拟机不想折腾了。。懒得装的，所以用Ubuntu for Win10来试试（在微软商店下载），然后 控制面板 > 程序 > 启动或关闭Windows功能
勾上适用于Linux的Windows子系统

下载LEDE K3 SDK

可以在windows下载, 也可以在ubuntu wget

中国科大提供了openwrt镜像和代理连接openwrt.proxy.ustclug.org
SDK地址:lede-sdk-17.01.4-bcm53xx_gcc-5.4.0_musl-1.1.16_eabi.Linux-x86_64

将源码下载到package目录下面

链接: https://pan.baidu.com/s/1pMPhJ5L 密码: fu6r

编译luci-app-syncdial

编译完成后文件在bin目录下面

安装

通过winscp上传路由器

然后进入路由器web管理页面就完事大吉

测试结果

我的宽带是20MB的, 电脑连接wifi所以没有达到下载三拨的慢速,上传已经是慢速了

MARK 软件包来自https://github.com/coolsnowwolf/lede
